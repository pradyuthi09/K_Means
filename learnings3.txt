Anomoly detection:

we use Isolation forest to detect otliers:The point no need much splitting to get leaf node that is the abnormal data point


#why required isolation there

1.Formula
       a.Anomaly score=1=Anomaly(strong,abonormal)
       b.Anomaly score=0.5(abnormal)
       c.Anomaly score=0=Normal Data point


       s(x,n)=z- E|h(x)|/c(n)

       h(x)=path length of data point
       E|h(x)|-avg length of data point
       c(n)=avg path of tree(no of edges from root to leaf)
               |
       2H(n-1)^-2(n-1)/n          h=harmonic number


       s(x,n)=2-E|h(x)/c(n)

       if E|h(x)| <<c(n)  
       anomoly-> score is close to 1(normal)
